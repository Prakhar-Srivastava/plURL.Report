  Executive Summary

Overview

Plurl  was engaged in an activity to conduct a URL Scan Security Assessment for malware detection.The purpose of the engagement was to utilise Plurl sandbox and AI in order to evaluate the site score for the given {{final_url}} and identify the vulnerabilities.

A Web Application Security scan Assessment provides Plurl with insight into the resilience of an application to withstand attack from malicious URL(s). The URL scan assessment evaluates the security of the application against best practice criteria to validate security mechanisms to identify vulnerabilities and malwares.

This report details the scope of scanning conducted, all significant findings along with a detailed report. The summary below provides a non technical audience with a summary of the key findings and relates these back to business impacts.

This document summarises the findings, analysis and recommendations from the assessment, which was conducted for {{final_url}}
